A comprehensive client management system for web development agencies.

== Description ==

TSWP Manage Web Clients is a powerful WordPress plugin designed specifically for web development agencies to manage their clients and associated services. The plugin provides a complete solution for tracking:

* Client information and contact details
* Domain registrations and expiration dates
* Web applications and hosting plans
* Email accounts and plans
* Payments and services

Key Features:

* Comprehensive client management
* Domain tracking and expiration alerts
* Application hosting plan management
* Email account management
* Payment tracking
* Service catalog management

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/tswp-manage-web-clients` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->TSWP Manage Web Clients screen to configure the plugin

== Frequently Asked Questions ==

= Can I export client data? =

Yes, the plugin provides export functionality for all client-related data.

= Is the plugin GDPR compliant? =

Yes, the plugin handles client data in accordance with GDPR guidelines.
